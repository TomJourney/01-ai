# 【1】概念介绍

1. 人工智能：<font color=red>指人类创造的计算机系统执行与人类智能相关的任务的能力。</font>如学习，推理，解决问题，感知及决策等能力。
   1. 人工指由人类创造或模拟；
   2. 智能：是智力与能力总称，指学习，推理，解决问题，感知及决策的能力；
2. agent（即智能体或智能代理）：<font color=red>指具有一定自主性的能够感知环境，做出决策并采取行动的系统</font>；
   1. agent的核心组件如下：
      1. 感知器：agent通过感知器收集环境信息，如通过传感器，数据库或互联网获取；
      2. 知识库：agent根据目标与以往经验，通过知识库存储和管理环境信息；
      3. 决策引擎：agent根据感知的信息并结合知识库中的数据，通过决策引擎做出决策；
      4. 执行器：agent通过执行器在环境中采取行动，如向机器人发出移动手臂指令，向web服务器发送消息；
3. 大模型：<font color=red> 指拥有数十亿甚至更多参数，利用海量数据通过深度学习训练得到的拥有学习，推理，解决问题，感知及决策等能力的模型</font>；
   1. 拥有通用推理能力的大模型，是推动agent从静态执行者向动态决策者转变的关键；
   2. 研究人员利用大模型构造agent的大脑；
   3. agent通过将大模型作为主要组件来扩展感知和行动空间，并通过策略和工具来制订具体的行动计划；
4. agent具备推理和规划能力：研究人员通过思维链（Chain of Thought， Cot），ReAct（Reasoning and Acting，推理并行动）和问题分解（Problem Decomposition）等逻辑框架，引导agent展现出强大的推理和规划能力；
