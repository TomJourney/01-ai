# 【1】RAG介绍

1. rag定义： retrieval augmented generation, 检索增强生成；

2. rag主要干两件事情：
   1. 从资料库检索相关内容；【<font color=red>先检索</font>】
   2. 就这些内容生成答案；【<font color=red>再生成</font>】
3. <font color=red>rag是目前最常用的AI问答方案之一</font>；
